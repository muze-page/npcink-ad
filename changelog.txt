=== Npcink Ad Changelog ===

== 0.3.3 ==

* Add a nonce-bound POST row action that duplicates one editable Promotion as a current-user draft.
* Copy native block content and only the explicitly accepted placement metadata while resetting publication state, dates, and schedule.
* Verify the created draft and every copied value, remove incomplete copies on failure, and report honestly when cleanup cannot be confirmed.
* Allow a native WordPress-scheduled Promotion to be paused from the list before it starts, while keeping Resume limited to reviewed drafts.
* Link overlap advisories to at most three exact Promotions and report any remainder without adding priority, rotation, or delivery changes.
* Align page-bar creative and dismissal controls to the active theme's wide content size, with a centered 1200px fallback for classic themes.
* Keep a 44 by 44 dismiss target while refining its visible surface to 32px and its WordPress close icon to 16px.
* Keep the single Promotion model, frontend delivery behavior, privacy boundary, and no-tracking contract unchanged.

== 0.3.2 ==

* Prepare the concise WordPress.org listing, localized directory images, review notes, and release checklist.
* Remove the unnecessary manual PHP text-domain loader and validate Simplified Chinese through the standard WordPress.org language-pack path.
* Fix the GitHub prerelease workflow repository context.
* Keep the single Promotion data model, live eligibility rules, and privacy boundary unchanged.

== 0.3.1 ==

* Increase the normal-flow page-bar dismiss target to 44 by 44 CSS pixels and reserve its space with RTL-safe logical padding.
* Refine the compact announcement pattern into wrapping short copy plus an editable core Button CTA.

* Align the GPLv2 license, WordPress.org contributor, public source, and reproducible build disclosures.
* Add official Plugin Check, classic/block theme compatibility, SHA-256 artifacts, and gated GitHub prerelease automation.
