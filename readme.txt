Magick AD helps you configure and manage ad placements with a modern admin UI.

== Description ==
Magick AD provides:

* Flexible ad placements and targeting
* Admin UI for configuring ads
* Diagnostics and reporting tools

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the WordPress Plugins screen.
